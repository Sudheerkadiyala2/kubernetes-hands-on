# Kubernetes Hands-on Project 🚀

## What I did

* Created Deployment using YAML
* Scaled Pods using replicas
* Exposed application using Service (NodePort)
* Used ConfigMap for environment variables
* Used Secrets for sensitive data

## Tech Used

* Kubernetes (Minikube)
* kubectl
* YAML

## Commands Used

```
kubectl apply -f deployment-config.yaml
kubectl apply -f service.yaml
kubectl apply -f configmap.yaml

```

## Outcome

Successfully deployed and managed a containerized application using Kubernetes with scaling  and
configuration.
